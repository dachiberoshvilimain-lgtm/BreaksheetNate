📋 Break Sheet — How It Works
Read this before you start. It covers everything.
⚡ First — Two Important Things
1 · It's fully live
We are all on the same screen. If you book a break — everyone sees it instantly. If you press Start or Stop — everyone sees it instantly. No saving. No refreshing. It just syncs automatically.
2 · Everything is Kyiv time
Use the Kyiv clock in the top right corner. Not your laptop. Not your phone. The screen clock.
🖥 What You're Looking At
The top bar shows the Kyiv clock in the middle, and your current shift + cutoff time on the right — so you always know how much time is left.

Below that is the Available Break Slots grid — the fastest way to see open times and book a break.

Below that is a policy reminder bar summarising the main rules.

The main table is split into two columns — Rows 1–5 on the left, Rows 6–10 on the right. One row per agent.

At the very bottom are live stats: who's on break right now, who's finished, planned breaks, and total agents loaded.
① Select Your Name
Pick any empty row (1–10). Click the dropdown that says — select agent —. Start typing your name to filter the list, then click it to select.

That row is now yours for this session.

If your name isn't there — tell a supervisor. We can add it instantly in the Agent Setup panel.
② Your Row Activates Automatically
As soon as you select your name, your row activates automatically — you'll see the blue outline appear straight away. No extra click needed.

You can also click anywhere on a row header to manually switch your active row — useful if you need to help someone else or correct a different row.

The habit is:
Select name → Row activates (blue outline) → Book break
③ Booking Your Break
You get 60 minutes total and up to 3 slots. Split however you want — 60, 30+30, 20+20+20, etc. Just don't go over 60 minutes.
🟢 Option A — Use the Grid (Fastest)
Each block in the grid = 15 minutes.

🟢 Green = Free  ·  🟠 Amber = 1 person booked (1 spot left)  ·  🔴 Red / crossed out = Full  ·  Grey = Time already passed  ·  Light blue NOW = Current window

Make sure your row is active. Then: 1st click = start time. 2nd click = end time. Done — it fills into your next empty slot automatically. If it breaks a rule, it flashes red. Just pick another time.
⌨️ Option B — Type Manually
Each slot has a start and end time box in HH:MM format. Type 2 digits for the hour — it jumps to minutes automatically. Type 2 digits for minutes, then press Tab or click away to save.

If valid → saves instantly. If not → turns red and clears. Try a different time.
④ Start & Stop Your Break
Booking a slot ≠ being on break.

Once both times are filled, a ▶ Start button appears. Press it when you're actually walking away — it shifts your break to begin right now while keeping the same duration.

Example: You booked 13:00–13:30. You press Start at 13:05. It becomes 13:05–13:35.

When you return, press ■ Stop. That locks the end time to right now and marks the slot as done.
🗑 Clear Buttons
Slot Clear — removes just that one break slot. Everything else stays.
Row Clear — wipes the agent name and all 3 slots for that row. Use if you need to start fresh.
Clear All Rows — resets every single row. Supervisors only. Used at the start of a new shift.
⚠️ Rules — Automatically Enforced
🍱 No breaks in the last 30 minutes of any shift. Blocked times: 14:30–15:00, 22:30–23:00, 06:30–07:00.
👥 Max 2 agents on break at the same time. A 3rd person cannot book into a full window.
⏱ Two agents can only overlap for up to 30 minutes. If agent A is on 05:00–06:00, agent B can only share 30 min of that (e.g. 05:30–06:00 is fine, 05:00–05:45 is not).
🕐 Max 60 minutes total per agent per shift, across all 3 slots combined.
⑤ Other Things to Know
🔴 Red input = time rejected (breaks a rule). Try a different time.
🌐 Everything syncs live — every change is visible to everyone instantly.
🕓 All times are Kyiv time — shown in the top-right clock. Use that, not your device.
➕ Name not in the list? Tell a supervisor. Open Agent Setup, type the name, press Save & Apply — syncs for everyone immediately.
