      📋 Break Sheet — How It Works
      Read this before you start. It covers everything.
      ⚡ First — Two Important Things
      1 · It's fully live
      We are all on the same screen. If you book a break — everyone sees it instantly. If you press Start or Stop — everyone sees it instantly. No saving. No refreshing. It just syncs automatically.
      2 · Everything is Kyiv time
      Use the Kyiv clock in the top right corner. Not your laptop. Not your phone. The screen clock.
      🖥 What You're Looking At
      The top bar shows the Kyiv clock and your current shift + cutoff time.

      Below that is the Available Break Slots grid — 🟢 Green = open slot, 🔴 Red / crossed out = already has someone booked there, Grey = time already passed.

      The main table is split into two columns — Rows 1–5 on the left, Rows 6–10 on the right.
      ① Select Your Name
      Pick any empty row (1–10). Click the dropdown that says — select agent —. Start typing your name to filter the list, then click it to select.

      That row is now yours for this session.
      ② Book Your Break
      You get 60 minutes total and up to 3 slots. Split however you want — 60, 30+30, 20+20+20, etc.
      🟢 Option A — Use the Grid (Fastest)
      Make sure your row is active. Then: 1st click = start time, 2nd click = end time. Done — fills into your next empty slot automatically.
      ⌨️ Option B — Type Manually
      Each slot has a start and end time box in HH:MM format. Type 2 digits for the hour — it jumps to minutes automatically.
      ③ Start & Stop Your Break
      Booking a slot ≠ being on break.

      Once both times are filled, a ▶ Start button appears. Press it when you're actually walking away.

      When you return, press ■ Stop. That locks the end time and marks the slot done.
      🗑 Clear Buttons
      Slot Clear — removes just that one break slot.
      Row Clear — wipes the agent name and all 3 slots for that row.
      Clear All Rows — resets every row. Supervisors only.
      ⚠️ Rules
      👤 Max 1 agent on break at the same time.
      🍱 No breaks in the last 30 minutes of any shift. Blocked times: 14:30–15:00, 22:30–23:00, 06:30–07:00.
      🕐 Max 60 minutes total per agent per shift, across all 3 slots combined.
